# shogi-board
対戦型の将棋盤です。

初めて自分で組んだプログラムになります。言語はjavascriptです。

使い方
ファイルのindex.htmlを開けば将棋盤を表示させられるようになっています。

ライブラリなど
ライブラリ・フレームワーク等は一切使ってません。そのうち使ってみたいです。
